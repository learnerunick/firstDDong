# Habit Tracker (로컬 스토리지 버전)

백엔드 없이 브라우저 `localStorage`에 데이터를 저장하는 습관 트래커 웹앱입니다.

## 지금 뭐부터 하면 좋을까?
아래 순서대로 시작하면 빠르게 MVP를 완성할 수 있어요.

1. **핵심 목표 정의**: "오늘 할 습관을 추가하고 체크/저장/삭제한다."
2. **MVP 완성**: 습관 추가, 완료 체크, 삭제, 로컬 저장 복원.
3. **사용성 보완**: 완료 항목 정리 버튼, 전체 초기화 버튼.
4. **확장 기능 추가**: 달력 뷰, 연속 달성(streak), 카테고리.

이 저장소에는 1~3단계가 구현되어 있습니다.

## 실행 방법
별도 설치 없이 `index.html`을 브라우저로 열면 동작합니다.

또는 로컬 서버로 실행할 수 있습니다:

```bash
python3 -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000` 접속.

## 현재 기능
- 습관 추가
- 완료 체크
- 개별 삭제
- 완료 항목만 일괄 정리
- 전체 초기화
- 새로고침 후에도 데이터 유지(localStorage)

## 파일 구조
- `index.html`: 화면 구조
- `style.css`: 스타일
- `app.js`: 상태 관리/렌더링/로컬스토리지 로직
